# Archie — Ronen Keisar brand kit

**Archie the beaver** is the brand mascot of Ronen Keisar: the architect you bring in when
something is broken or missing and needs to be built, scaled, and made to stick.

**Motto:** See it whole. Build it right. Keep it honest.

## Live template builder
Enable GitHub Pages (Settings → Pages → Deploy from branch → main, root) and open the builder
at the published URL. Configure variant, crop, lockup, and background; copy or download
export-ready SVG.

## Contents
- `index.html` — interactive template builder (self-contained)
- `svg/` — vector masters: primary, dark-background (off-white / gold ring), single-color (navy / ink), icon crop, wordmark lockups ("Ronen Keisar", horizontal/stacked, motto optional)
- `png/` — raster exports 48–1024 px + 2048 px source
- `proofs/` — QA sheets

## Locked tokens
Ring navy `#002B7F` · Hat gold `#C8A24B` · Fur `#985922` / `#7A4A20` / `#5A3414` ·
Muzzle `#EBAB5E` / `#EFC28A` · Freckle `#C49256` · Outline ink `#00183C` ·
Type: Work Sans 700 (wordmark), motto in Work Sans 400.

## Rules (short form)
Clear space = 1 ring-width. Minimum 48 px (64 px recommended). No gradients, no scene
elements, no text baked into the mark. Never navy ring on dark backgrounds — use the dark
variants. Lockup text: "Ronen Keisar", motto optional.
